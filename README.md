## 概要

以下の2つの機能を満たすための業務用スクリプトを準備しています。

- blockscoutv5から取得される特定のaddressのtokenTransferデータを修正するスクリプト
- 特定のaddressのcommision rewardによって生じたお金の記録を出力するスクリプト
