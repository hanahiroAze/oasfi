# for developer

## 実行ファイル作成

* nodeはv18を利用してください
* eslintとprettierは実行しないでください

```bash
nvm use 18
git clone https://github.com/oasysgames/explorer-csv-cli.git
cd explorer-csv-cli
npm i
npm i -g pkg
npm run compile
```

* 上記六つのコマンドを打つとローカルに実行ファイルが生成されます。あとは

- [ユーザー向けガイド](./user_guide.md)

を参照してください。
