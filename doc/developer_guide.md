# for developer

## create execution file

* please use node v18

```bash
nvm use 18
git clone https://github.com/oasysgames/oasfi.git
cd oasfi
npm i
npm i -g pkg
npm run compile
```

* Executing the above six commands will generate the executable file locally. After that, you can...

- watch [ユーザー向けガイド](./user_guide.md)
